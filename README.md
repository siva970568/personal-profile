### Step 1: Install Git
1.  Download Git from [git-scm.com](https://git-scm.com/downloads).
2.  Install it (you can click "Next" through the default options).
3.  Open a new terminal (Command Prompt or PowerShell).

### Step 2: Create a Repository on GitHub
1.  Go to [github.com/new](https://github.com/new).
2.  Repository name: `personal-profile` (or whatever you like).
3.  Description: "My professional portfolio website".
4.  Click **Create repository**.

### Step 3: Upload Your Code
Open your terminal in this project folder (`C:\Users\hraog\.gemini\antigravity\scratch\personal_profile`) and run these commands one by one:

```bash

# 2. Configure your identity (if you haven't already)

# 4. Commit your changes

# 5. Link to your GitHub repository
git remote add origin https://github.com/siva970568/personal-profile.git

# 6. Push the code
git branch -M main
git push -u origin main
```
